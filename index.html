Case Conversion Guide

Programming Cases

  • camelCase: `myVariableName` - Standard for JavaScript/Java variables.
  • snake_case: `my_variable_name` - Common in Python and databases.
  • kebab-case: `my-variable-name` - Used in URLs and CSS classes.
  • PascalCase: `MyVariableName` - Used for classes in C#, Java, JS.

Writing Styles

  • Title Case: Capitalizes major words. Perfect for headlines.
  • Sentence case: Only the first letter is capitalized. Natural flow.
  • UPPERCASE: Emphasizes text, often used for warnings or acronyms.
